# Aula 1: Engenharia de Prompt
Práticas de Engenharia de Prompt aplicada à área administrativa:
• Clareza, especificidade e estrutura lógica
• Uso de contexto institucional
• Definição de persona técnica (analista, gestor, auditor)
• Exemplos estruturados para relatórios e pareceres
• Prompt em múltiplas etapas para análise administrativa

Exploração prática de IA generativa aplicada à:
• Elaboração de relatórios
• Criação de pareceres técnicos
• Planejamento administrativo assistido por IA
